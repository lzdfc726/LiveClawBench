# Microservice Stack Deploy

Your email inbox (http://localhost:5174/) contains a message from the Tech Lead.

## Objective

The source code for multiple microservices under `/workspace/services/` needs to be deployed as a complete service stack. The services have interdependencies and must be started in the correct order with proper inter-service communication configured.

## Deliverables

1. All services running and able to communicate with each other
2. End-to-end data storage and retrieval accessible through the frontend page
3. The entire service stack accessible via HTTP
