# DeFi Ecosystem Architecture Overview

## Core Components

DeFi (decentralized finance) is built on blockchain technology and mainly includes the following components:

### 1. Smart Contracts
Self-executing contract code running on public chains like Ethereum.

### 2. DEX (Decentralized Exchanges)
Such as Uniswap and Curve, allowing users to swap tokens without intermediaries.

### 3. Lending Protocols
Such as Aave and Compound, supporting over-collateralized lending.

### 4. Stablecoins
Such as USDC and DAI, providing price stability.

### 5. Yield Aggregators
Such as Yearn Finance, automatically optimizing yield strategies.

## Basic Operation Mechanism

Users interact with smart contracts through wallets (e.g., MetaMask). All transactions are recorded on the blockchain publicly and transparently.
