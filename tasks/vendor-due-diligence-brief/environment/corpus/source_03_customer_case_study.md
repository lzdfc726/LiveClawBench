# Customer Case Study — MedRecord Health Systems

*Published by CloudEdge Systems, February 2026*

## Background

MedRecord Health Systems is a mid-sized US healthcare provider managing electronic health records for 220,000 patients. In Q2 2025, MedRecord engaged CloudEdge Systems to encrypt their data pipeline between their EHR application and their cloud storage tier.

## Deployment

CloudEdge's DataGuard middleware was deployed as a Kubernetes sidecar across MedRecord's AWS EKS cluster. The integration took 8 weeks end-to-end (within the 9-week estimate).

## Outcomes

- 100% of PHI data encrypted at the application layer before cloud storage
- HIPAA audit passed in September 2025 with no data security findings
- Average API latency overhead from DataGuard: 12ms (within the 15ms SLA)
